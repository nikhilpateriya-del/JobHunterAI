# JobHunterAI 🚀

An AI-powered Remote Job Search Assistant built using Python.

## Features

- Fetch jobs from multiple sources
- Filter Data Analyst jobs
- AI Resume Matching
- ATS Score
- AI Cover Letter
- Telegram Notifications
- Google Sheets Integration

## Tech Stack

- Python
- Pandas
- Requests
- Playwright
